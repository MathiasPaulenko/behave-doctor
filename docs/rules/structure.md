# Structure Rules (BD101-104)

Informational metrics about the project. These rules never fail the build
(severity `info`) and exist to give you a quick overview of suite size.

## BD101 — feature-count

Reports the total number of features in the project.

| Attribute   | Value          |
| ----------- | -------------- |
| Severity    | info           |
| Category    | Structure      |
| Configurable| No             |

## BD102 — scenario-count

Reports the total number of scenarios across all features.

| Attribute   | Value          |
| ----------- | -------------- |
| Severity    | info           |
| Category    | Structure      |
| Configurable| No             |
